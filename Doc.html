Build Instruction:

Here we’ve used a third party json parser for C++. You can find the repository here.

Integration :

json.hpp is the single required file in single_include/nlohmann or released here. You need to add :

#include <nlohmann/json.hpp>

// for convenience

using json = nlohmann::json;

to the files you want to process JSON and set the necessary switches to enable C++11 (e.g., -std=c++11 for GCC and Clang).

  1. bits/stdc++.h to avail all the necessary c++  std:: header files.

#include <bits/stdc++.h>

// for convenience

using namespace std;
